# UI Components

This folder contains reusable UI components for the Timeline application.

## Guidelines

- All components should be built using DaisyUI components and Tailwind CSS
- Components should be TypeScript-based with proper type definitions
- Follow the project's code style guidelines (2-space indentation, single quotes)
- Use the `cn()` utility from `@/lib/utils` for conditional class merging

## Available DaisyUI Components

DaisyUI provides ready-to-use components including:

- Buttons
- Cards
- Modals
- Forms (inputs, selects, textareas)
- Navigation (navbar, menu, tabs)
- Badges
- Alerts
- And more...

Reference: https://daisyui.com/components/
